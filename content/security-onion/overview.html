What is Security Onion?

Security Onion is a free, open-source Linux distribution for threat hunting, enterprise security monitoring, and log management. It bundles multiple NSM tools into a single platform.

Core Components

ComponentRole
SuricataIDS/IPS — generates alerts from rule signatures
Zeek (Bro)Network analysis — generates structured connection logs
ElasticsearchLog storage and indexing backend
KibanaWeb UI for querying and visualizing logs
LogstashLog ingestion and normalization pipeline
StenographerFull packet capture (PCAP) storage
OSSEC/WazuhHost-based intrusion detection (HIDS)

Security Onion Console (SOC) — Key Sections

SectionURL PathWhat It Shows
Alerts/alertsSuricata IDS alerts — primary triage view
Dashboards/dashboardsPre-built visualizations (DNS, HTTP, alerts)
Hunt/huntThreat hunting interface — query logs
PCAP/pcapPacket capture retrieval and analysis
Cases/casesCase management for incidents
Grid/gridNode management and health status

Deployment Types

TypeUse Case
StandaloneAll components on one node — lab/small org
Manager + SensorSensor captures traffic, manager holds data
DistributedMultiple sensors reporting to cluster — enterprise
