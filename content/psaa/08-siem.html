08 · SIEM

What a SIEM Does

  • Log Aggregation — collects logs from ALL sources into one place
  • Normalization — converts different log formats into a standard format
  • Correlation — links related events across systems to find attack patterns
  • Alerting — fires when rules are matched
  • Dashboards — visual overview of security posture
  • Retention — stores logs for compliance and investigation

Splunk — SPL Basics

# Basic search
index=main sourcetype=access_combined

# Count events by field
index=main | stats count by src_ip

# Top 10 source IPs
index=main | top limit=10 src_ip

# Time chart of events
index=main | timechart count

# Search for failed logins
index=main "Failed password" | stats count by src_ip | sort -count

Elastic KQL Basics

# Match field value
event.action : "logon-failed"

# AND condition
event.action : "logon-failed" AND source.ip : "10.0.0.*"

# Wildcard search
process.name : "power*"

# Exists check
destination.port : 4444
