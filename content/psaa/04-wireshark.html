Wireshark Tips

  • Statistics > Conversations — see who is talking to whom and how much
  • Statistics > Protocol Hierarchy — breakdown of traffic types
  • File > Export Objects > HTTP — extract files transferred over HTTP
  • Follow TCP Stream — see the full conversation between two hosts
  • Right click any packet > Apply as Filter — instant filtering
  • Edit > Find Packet > String — search packet contents for keywords

Wireshark — Investigation Workflow

  1. Open pcap → check Statistics > Conversations to find top talkers
  2. Check Protocol Hierarchy for unexpected protocols
  3. Filter by suspicious IP or port
  4. Follow TCP Stream to read content
  5. Export HTTP objects if files were transferred
  6. Hash any extracted files for VirusTotal lookup

tcpdump Quick Reference

# Capture to file for later Wireshark analysis
sudo tcpdump -i eth0 -w capture.pcap

# Read existing capture
sudo tcpdump -r capture.pcap

# Combine filters with and/or
sudo tcpdump -i eth0 src 10.0.0.1 and port 443

# Show as ASCII (useful for HTTP)
sudo tcpdump -i eth0 -A port 80

# No hostname resolution (faster, cleaner)
sudo tcpdump -i eth0 -nn port 22
