What EDR Monitors

  • Process creation and termination
  • Network connections initiated by processes
  • File creation, modification, deletion
  • Registry changes (Windows)
  • User logon/logoff events
  • Script execution (PowerShell, bash)
  • Memory injection attempts
  • Lateral movement indicators

EDR vs Antivirus

FeatureTraditional AVEDR
Detection MethodSignature-based onlyBehavior + signatures + AI
ResponseQuarantine fileIsolate host, kill process, rollback
VisibilityFile-levelFull process tree, network, registry, memory
Threat HuntingNoYes — query historical data
ForensicsLimitedFull timeline and event correlation
