03 · Network Security Monitoring (NSM)

Packet / Frame Structure

Every packet travels through layers — FRAME → IP → TCP → DATA

LayerHeader ContainsFocus On
FRAME (Layer 2)Source MAC, Destination MACWho is on the local network
IP (Layer 3)Source IP, Destination IPWhere packet is going
TCP/UDP (Layer 4)Source Port, Destination PortWhat service/application
Data (Layer 7)Payload (index.html, etc)What was actually sent
KEY: Always focus on SOURCE ADDRESS and DESTINATION ADDRESS first.

tcpdump — Essential Commands

# Capture all traffic on interface
sudo tcpdump -i eth0

# Save capture to file
sudo tcpdump -i eth0 -w capture.pcap

# Read saved pcap file
sudo tcpdump -r capture.pcap

# Filter by host IP
sudo tcpdump -i eth0 host 192.168.1.1

# Filter by port
sudo tcpdump -i eth0 port 80

# Filter by source IP
sudo tcpdump -i eth0 src 10.0.0.1

# Filter by destination IP
sudo tcpdump -i eth0 dst 10.0.0.2

# Show ASCII readable output
sudo tcpdump -i eth0 -A

# Show hex and ASCII
sudo tcpdump -i eth0 -XX

# Capture only N packets
sudo tcpdump -i eth0 -c 100

# Capture HTTP and HTTPS
sudo tcpdump -i eth0 port 80 or port 443

# Don't resolve hostnames (faster)
sudo tcpdump -i eth0 -n
